Project Description
Arcade Shooter Game is a simple 2D game developed using Godot. The player controls a spaceship that continuously shoots at enemies.
Game mechanics:
New enemies appear when the player reaches certain scores
Enemy spawn rate increases progressively as the game advances
The objective is to survive as long as possible and achieve the highest score
This project is intended as a programming exercise to practice game logic, player controls, and dynamic enemy spawning.


How to Run
Install Godot Engine
Download or extract the game ZIP
Open the project in Godot


Project Structure
main.tscn → main scene of the game
assets/ → folder containing sprites, fonts, and other resources
